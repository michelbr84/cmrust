CM Rust - Windows Release
===========================

Instructions:
1. Double-click "cm_tui.exe" to play the game in Terminal UI mode (Recommended).
2. For CLI usage, open a terminal in this folder and run "cm.exe --help".

Files:
- cm_tui.exe: Main game executable (Text-based interface)
- cm.exe: Command-line utility for additional tasks
- assets/: Game data files (Required)

Enjoy!
